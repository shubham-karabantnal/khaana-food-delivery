# 🍲 Khaana – Food Ordering Platform

Khaana is a full-stack food ordering and restaurant management web application built to deliver a seamless and modern user experience. It connects customers with restaurants through an intuitive UI and a robust backend system.

---

## 🚀 Features

- 🔐 **Role-Based Authentication**
  - Separate login/register for Customers & Restaurant Owners
  - Secure OTP-based authentication

- 🍽️ **Restaurant Discovery**
  - Browse restaurants with ratings, location, and details
  - Clean and responsive UI

- 📜 **Interactive Menus**
  - Categorized menu items with descriptions & pricing
  - Easy add/remove controls for cart

- 🛒 **Real-Time Cart System**
  - Dynamic pricing and order summary
  - Delivery fee calculation

- 🧾 **Automated Billing**
  - PDF bill generation after order placement

- ⚡ **Optimized Backend**
  - Raw SQL queries & stored procedures for performance
  - Efficient transaction handling

---

## 🛠️ Tech Stack

**Frontend:**
- React.js

**Backend:**

**Database:**

**Other Integrations:**
